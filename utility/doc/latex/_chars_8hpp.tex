\subsection{Detailed Description}
A simple class to encapsulate char $\ast$ handling. 

\begin{DoxyAuthor}{Author}
(c) 1999-\/2001 Frank Hoeppner \href{mailto:frank.hoeppner@ieee.org}{\tt frank.\+hoeppner@ieee.\+org}
\end{DoxyAuthor}
We implement a very simple {\ttfamily \hyperlink{class_chars}{Chars}} template class for {\ttfamily char}\mbox{[}n\mbox{]} variables. If you need dynamic and more complicated string handling, use the S\+T\+L {\ttfamily string} class. Here, we just try to encapsulate the various {\ttfamily strcmp}, {\ttfamily strncpy} ... C functions to handle character strings of type {\ttfamily char$\ast$}. We define the common operators for string concatenation, comparison, etc. Function names are related to the names used in the S\+T\+L string class.
