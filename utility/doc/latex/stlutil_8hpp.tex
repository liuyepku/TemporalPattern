\subsection{Detailed Description}
Implementation of \hyperlink{structstring__less}{string\+\_\+less}. 

This file provides a couple of functors which are helpful when using the standard template library (S\+T\+L). To use a S\+T\+L \mbox{[}\mbox{[}set$<$\+T,...$>$\mbox{]}\mbox{]} or \mbox{[}\mbox{[}map$<$\+T,...$>$\mbox{]}\mbox{]} data structure, one must provide a \mbox{[}\mbox{[}binary\+\_\+function$<$\+T,\+T,bool$>$\mbox{]}\mbox{]} to compare the \href{resp.\
[[map]]}{\tt \mbox{[}set\mbox{]}} contents. We define some of those binary functions for frequently used types \mbox{[}\mbox{[}T\mbox{]}\mbox{]}. Furthermore, we present some composition functors to compose binary functions.
