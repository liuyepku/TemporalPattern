Detailed Description

Binary function to compare char* pointer via strcmp.

The comparison yields true, if the first comes lexicographically before the second (strcmp returns -1 ). The function handles the special pointers NULL and 0xFFFF = ~0 as top and bottom elements, that is: a NULL string comes before any other string, a ~0 string after any other string.
